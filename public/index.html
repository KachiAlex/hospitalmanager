Hospital Management API

Lightweight backend to onboard patients, manage encounters, issue lab orders and prescriptions, and control admissions with bed assignments.

Quick Start

npm install
npm start
# Base URL: http://localhost:3000

Key Endpoints

  • GET /health – service heartbeat
  • POST /patients – register patient
  • POST /doctors – add doctor profile
  • POST /encounters – doctor visit w/ notes
  • POST /lab-orders & PATCH /lab-orders/:id – manage labs
  • POST /prescriptions – medication plan
  • GET /beds – ward/bed availability
  • POST /admissions & /admissions/:id/discharge – bed workflow

Sample Flow

curl -X POST http://localhost:3000/doctors \ 
  -H "Content-Type: application/json" \
  -d '{"firstName":"Ada","lastName":"Byron","specialty":"Cardiology"}'

curl -X POST http://localhost:3000/patients \ 
  -H "Content-Type: application/json" \
  -d '{"firstName":"John","lastName":"Doe","gender":"male","dateOfBirth":"1980-04-01"}'

curl -X POST http://localhost:3000/encounters \ 
  -H "Content-Type: application/json" \
  -d '{"patientId":1,"doctorId":1,"visitDate":"2025-12-30"}'
